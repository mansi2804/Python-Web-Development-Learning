About Python Shop

Our Mission

Python Shop is a demonstration project showing how to integrate front-end technologies with Flask. It showcases various techniques for building modern web applications using Python.

Key Learning Concepts

Flask uses the Jinja2 templating engine to render HTML dynamically. This allows for:

  • Template inheritance with {% raw %}{% extends %}{% endraw %}
  • Variable interpolation with {% raw %}{{ variable }}{% endraw %}
  • Control structures like {% raw %}{% if %}{% endraw %} and {% raw %}{% for %}{% endraw %}
  • Custom filters and functions to format data

This project demonstrates all these features through a practical e-commerce example.

Flask serves static files (CSS, JavaScript, images) from the static directory. Key points:

  • Use {% raw %}{{ url_for('static', filename='path/to/file') }}{% endraw %} to generate URLs
  • Files are organized by type: CSS, JavaScript, images
  • Static files are cached by browsers for performance
  • In production, consider using a CDN for better performance

This project demonstrates how to handle HTML forms in Flask:

  • Creating forms with proper method attributes (GET/POST)
  • Processing form data with request.form
  • Form validation and error handling
  • Redirecting after form submission to prevent duplicate submissions
  • Using Flash messages to provide user feedback

Check out the Contact and Add/Edit Product forms for examples.

The application uses Bootstrap 5 for responsive design:

  • Mobile-first approach ensures the site works on all devices
  • Grid system adapts content based on screen size
  • Components like navigation, cards, and forms are responsive
  • Custom CSS enhances the Bootstrap defaults

Try resizing your browser window to see the responsive design in action!
