Welcome to Flask Templates Demo

This page demonstrates basic variable passing from Python to templates.

Template Variables:

  • Current Time: {{ current_time.strftime('%Y-%m-%d %H:%M:%S') }}
  • Random Number: {{ random_number }}

Jinja2 Expressions:

  • Random Number + 10: {{ random_number + 10 }}
  • Random Number * 2: {{ random_number * 2 }}
  • Is Random Number > 50? {{ 'Yes' if random_number > 50 else 'No' }}

Filters Example:

  • Title Case: {{ "hello world"|title }}
  • Uppercase: {{ "hello world"|upper }}
  • Length of "Flask": {{ "Flask"|length }}
