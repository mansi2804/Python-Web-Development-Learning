How Template Inheritance Works

The {% extends "base.html" %} statement at the top of this template tells Jinja2 that this template "extends" the base template.

We can then define blocks that override the blocks defined in the parent template:

  1. The content block replaces the default content in the base template
  2. We could also override the footer block if needed
  3. Any blocks not overridden will use the parent template's version

This allows for consistent layouts across your application while keeping templates DRY (Don't Repeat Yourself).
